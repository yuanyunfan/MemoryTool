## Architecture Notes

- **Shared SQLite**: GUI and MCP server are separate processes sharing one .db file via WAL mode
- **MCP Transport**: stdio only (V1). Claude Code spawns MemoryMCP as subprocess
- **No print() in MCP target**: stdout is MCP protocol channel, use os_log/stderr for debugging
- **Swift MCP SDK**: official modelcontextprotocol/swift-sdk 0.12.0
- **DB Library**: GRDB.swift for type-safe SQLite access with FTS5 support
